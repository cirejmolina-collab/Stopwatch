# Advanced Stopwatch App

A feature-rich stopwatch web app with lap tracking, session saving, dark mode, and more. Perfect for timing activities, workouts, or productivity sessions.

## Features
- Start, pause, and reset the stopwatch
- Record unlimited laps
- Save sessions with lap details to local storage
- View saved session history
- Clear laps without resetting the timer
- Dark mode toggle
- Responsive, modern UI

## Instructions to Run the Project
1. Place all files (`index.html`, `style.css`, `script.js`) in the Stopwatch folder.
2. Open `index.html` in your browser.
3. Use the controls to start, pause, reset, record laps, and save sessions.
